Pizza From SpongeBob

Limits: 3s, 1.0 GB

SpongeBob recently got obsessed with his (not perfect) rectangular face and has started making rectangular M x N grid like pizzas for his customers. Yes, he sells pizzas nowadays. If anyone can tell him how many unique rectangles can be found in this M x N grid (rectangle’s corners have to be any of the integer points within the grid), then SpongeBob will serve them that pizza for free!

Now you all know this brainless friend of SpongeBob, Patrick, who just forgot where he keeps his money, wants to have free pizzas from Sponge. Help Patrick to solve this challenge.

Input

You will be given C, the number of pizzas Patrick wants to have followed by C lines. Each line will contain two numbers: M and N, the size of each pizza.

Constraints are:

1 ≤ C ≤ 106
1 ≤ M, N ≤ 3*103

Output

Print one line for each pizza containing the answer of the challange.

Samples

Input Output 
3
2 2
3 3
10 20

10
44
18841

Explanation for the first case:

There are 9 orthogonal and 1 diagonal (red colored) rectangles.
