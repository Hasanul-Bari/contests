National ICT Day Contest 2018

Noogler Mamun & his Adjophobia

Limits: 1s, 1.0 GB

Noogler Mamun is in emergency. But fortunately his phone has an emergency system. He has to type his emergency word on his phone and local police will know his location instantly. As he is Googler, he likes privacy, so he uses an old generation phone.

But here comes the problem. Mr Mamun has a disease named adjophobia. He can type a letter if it is adjacent to his previous button pressed, here adjacent means if it shares a side or the same letter. He can choose his first letter on his wish.

Mr Mamuns phone is intelligent enough that if he types a letter wrong, it will automatically erase that. But you can only use the numbers 2 to 9 in the keypad

Now we have to save Mr Mamun. Lets find him the path to minimum number of key presses.

Input

First line of the input contains number of test cases, t. For each test case, there will be a string s, which is actually the emergency word. Emergency word will contain only the small english letters [a-z].

1 <= t <= 100000 1 <= len(s) <= 10000 total sum of length of the all test cases <= 10000000

Output

There will be two lines for each test case. On the first line, print the minimum number of presses to type the emergency word. And on the second line, print the path to get the emergency word. If there is multiple answers, print the lexicographical smallest one.

Samples

Input Output 
3
a
ab
abc
